# Truec
This App will fetch the HTML content from a webpage and perform the string manipulation.

Componets and technologies used in this app.
* Kotlin
* Coroutines
* RxJava
* ScalarsConverterFactory(for HTML parsing)
* Espresso (UI testing)
* junit
